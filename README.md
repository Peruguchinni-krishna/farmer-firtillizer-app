# 🌾 Farmer Fertilizer App (FertiGuide)

An AI-powered web application that helps farmers choose the **right fertilizer** based on soil nutrients, crop type, and environmental conditions.

---

## 🚀 Project Overview

Farmers often struggle to select the correct fertilizer due to lack of knowledge about soil composition and crop requirements. This application provides **smart fertilizer recommendations** to improve crop yield and reduce chemical overuse.

---

## 🎯 Objectives

* Recommend the most suitable fertilizer
* Improve crop productivity
* Reduce excessive chemical usage
* Provide an easy-to-use digital tool for farmers

---

## 🧠 Features

### ✅ Core Features

* Input crop type and soil type
* Enter NPK values (Nitrogen, Phosphorus, Potassium)
* Add temperature and humidity
* Get instant fertilizer recommendation

### 🚀 Advanced Features

* 📊 Smart prediction logic
* 📱 Installable as a mobile app (PWA)
* 🌐 Works offline using Service Worker
* 📈 Dashboard with recent predictions
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **PWA:** Manifest.json + Service Worker

## ▶️ How to Run the Project

1. Open the project in VS Code
2. Install **Live Server extension**
3. Right-click `index.html` → **Open with Live Server**
4. Open in browser:

   ```
   http://127.0.0.1:5500
   ```

---

## 📲 How to Install as App (PWA)

### 💻 On Laptop

* Open app in Chrome
* Click **Install App (⬇️ icon)** in address bar

### 📱 On Mobile

* Open app using your local IP
* Tap **3 dots → Add to Home Screen**
